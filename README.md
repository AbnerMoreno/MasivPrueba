# pruebaMasiv


## Iniciamos abriendo la consola y escribiendo lo siguiente: 

```sh
npm install
```

## Una vez instalado todos los paquetes, procedemos abrir dos terminales en una escribiremos ---->

```sh
npm run dev
```
## Y en la otra ----->

```sh
npm run build
```
